Addon to enable 1 macro status removal with the help of [send](../send/).

## Priority

1. Doom
2. Curse
3. Petrification
4. Paralysis
5. Plague
6. Silence
7. Blindness
8. Poison
9. Diseased

## Commands

The only command is `stna`. Using this command will remove the debuffs above in the priority listed. Also will use holy waters when doomed.
